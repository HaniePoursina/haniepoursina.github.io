ioFog

ioFog is an edge computing platform for deploying, running, and networking distributed microservices at the edge. In this project an Edge Computing platform was implemented on three Raspberry Pis and they communicated over network providing by Kubernetes Cluster and software deployed by Docker.

Variational Auto Encoders

Variational AutoEncoder is one of the Generative models that have been introduced in recent years. This project is a practice of using it for voice generation.
Dataset has taken from kaggle, Project was tested on colab networks

Weather Forcasting

This project involved 10,000 data of weather of some places, which aimed to predict which weather is for where. I used KNN algorithm and PCA to reduce dimensions and reached 90% accuracy by using test data splitted from trains.
